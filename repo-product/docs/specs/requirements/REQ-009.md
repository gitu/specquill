# External source importers

Non-git reference knowledge (web pages, API contracts, wikis) SHALL be
mirrored into read-only repositories the copilot can ground on.

> **REQ-009.1 · MUST** — A source of kind `url`, `openapi` or `confluence`
> SHALL be materialized into a read-only mirror repository by an importer.

> **REQ-009.2 · MUST** — Importer credentials SHALL come only from the
> environment (via a configured variable name); they SHALL NOT appear in
> command lines, config files or the database.

> **REQ-009.3 · MUST** — An import SHALL be an idempotent full snapshot:
> unchanged upstream content SHALL NOT create a new commit.

> **REQ-009.4 · SHOULD** — Imports SHALL run on a schedule and on demand, and
> the last import's outcome SHALL be reported to administrators.
