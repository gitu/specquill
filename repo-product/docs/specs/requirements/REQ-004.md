# Multi-stage source authorization

Reference sources (regulation texts, other repos, code) power AI grounding —
but selecting them must never mint access.

> **REQ-004.1 · MUST** — Credentials and remote URLs SHALL only be defined in
> the app catalog (stage 1); credentials come from the environment, never
> from the database or any in-repo file.

> **REQ-004.2 · MUST** — Tenant administrators SHALL grant catalog sources to
> the tenant (stage 2) before any project can use them.

> **REQ-004.3 · MUST** — The in-repo project config SHALL only select granted
> sources by name (stage 3); ungranted selections are surfaced as warnings
> and never fetched.

> **REQ-004.4 · MUST** — Write operations SHALL require at least the member
> role; tenant administration requires the admin role (stage 4).
