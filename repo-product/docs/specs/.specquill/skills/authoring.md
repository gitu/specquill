# Skill: authoring in this workspace

- These documents specify SpecQuill itself — keep them true to the shipped
  behavior; when code and spec disagree, file a change record before editing.
- RFC-2119 keywords only inside requirement statements.
- Reference implementation invariants by their doc anchor, e.g.
  specs/content-root.md#mapping-rules.
