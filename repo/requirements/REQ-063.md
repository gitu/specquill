# Partial-fill Reporting

When an order is executed across multiple partial fills, each fill SHOULD be reportable
individually while rolling up to the parent order.

> **REQ-063.1 · SHOULD** — The trade model SHALL carry a `fills[]` array; each fill
> SHALL reference its parent `trade.id`.

## Drivers
- ◆ **Product** — Product manager request (2026-06). *No regulatory driver.*

This is a purely product-driven requirement — included to show the workspace tracks
non-regulatory sources alongside regulatory ones.
